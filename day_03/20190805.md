행우선: i for 문 돌리고 j

열우선: j for 문 돌리고 i

지그재그: index 값이 짝수이면 왼우 홀수이면 우왼

```python
# 지그재그 순회
N =10

for i in range(N):        #i = 행
	if i % 2 == 0:
		for j in range(N):  # j = 열
			pass
	else:
		for j in range(N-1, -1, -1):
			pass
```
